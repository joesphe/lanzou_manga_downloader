# 蓝奏云漫画下载器 - 精简版

这是一个打包好的蓝奏云漫画下载器独立程序，无需安装Python环境即可运行。

## 程序信息

- **文件名**: 蓝奏云漫画下载器_精简版.exe
- **文件大小**: ~21MB
- **功能**: 专用于处理预设的蓝奏云链接
- **运行环境**: Windows 64位系统

## 使用说明

1. 双击 `蓝奏云漫画下载器_精简版.exe` 启动程序
2. 程序启动后会显示当前处理的预设链接信息
3. 点击"获取文件列表"按钮获取文件
4. 选择要下载的文件（单击选择单个，Ctrl+单击选择多个，Shift+单击选择范围）
5. 设置下载目录（默认为downloads文件夹）
6. 点击"下载选中文件"或"下载全部"开始下载
7. 在进度条区域查看下载进度
8. 在日志区域查看操作记录

## 注意事项

- 首次运行可能需要等待一段时间，因为程序需要初始化浏览器环境
- 如果杀毒软件提示风险，可以暂时关闭实时保护或将程序加入白名单
- 程序会自动使用无头模式运行，不会弹出浏览器窗口
- 下载的文件会保存在指定的下载目录中

## 常见问题

**Q: 程序启动很慢怎么办？**
A: 首次启动需要初始化浏览器环境，后续启动会更快。

**Q: 为什么没有看到浏览器窗口？**
A: 程序默认使用无头模式运行，以提供更好的用户体验。

**Q: 下载失败怎么办？**
A: 检查网络连接，或稍后再试。程序会自动重试3次。
