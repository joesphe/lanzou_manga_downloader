# 蓝奏云下载器 - 发布包

## 包含的程序

### 1. 蓝奏云漫画下载器_精简版.exe
- **类型**: GUI图形界面版本
- **用途**: 给最终用户使用
- **特点**: 
  - 无控制台窗口
  - 专门用于预设链接
  - 隐藏复杂设置
  - 提供友好的图形界面

### 2. 蓝奏云下载器_调试版.exe
- **类型**: 命令行界面版本
- **用途**: 用于调试和开发
- **特点**:
  - 有控制台输出
  - 显示详细日志信息
  - 便于排查问题
  - 非无头模式运行（可以看到浏览器）

## 使用说明

### 精简版 (GUI)
1. 双击 `蓝奏云漫画下载器_精简版.exe` 启动程序
2. 程序启动后会显示当前处理的预设链接信息
3. 点击"获取文件列表"按钮获取文件
4. 选择要下载的文件（单击选择单个，Ctrl+单击选择多个，Shift+单击选择范围）
5. 设置下载目录（默认为downloads文件夹）
6. 点击"下载选中文件"或"下载全部"开始下载
7. 在进度条区域查看下载进度
8. 在日志区域查看操作记录

### 调试版 (CLI)
1. 双击 `蓝奏云下载器_调试版.exe` 启动程序
2. 程序会在控制台窗口中显示详细信息
3. 按照提示选择操作：
   - `all`: 下载所有文件
   - `first`: 下载第一个文件
   - `quit`: 退出程序
4. 观察控制台输出以了解程序运行情况

## 注意事项

- 首次运行可能需要等待一段时间，因为程序需要初始化浏览器环境
- 如果杀毒软件提示风险，可以暂时关闭实时保护或将程序加入白名单
- 程序会自动使用内置浏览器引擎
- 下载的文件会保存在指定的下载目录中

## 常见问题

**Q: 程序启动很慢怎么办？**
A: 首次启动需要初始化浏览器环境，后续启动会更快。

**Q: 下载失败怎么办？**
A: 检查网络连接，或稍后再试。程序会自动重试3次。

**Q: 如何查看详细的运行日志？**
A: 使用调试版程序，它会在控制台显示详细信息。
