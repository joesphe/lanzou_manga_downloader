# 蓝奏云下载器 (Lanzou Downloader)

一个基于Selenium的蓝奏云文件下载工具，支持Chrome和Edge浏览器，可实现免登录批量下载蓝奏云分享链接中的文件。

## 功能特性

- 支持Chrome和Edge浏览器
- 免登录下载蓝奏云文件
- 并发下载提高效率
- 自动处理密码验证
- 文件名合法性检查和修复
- 详细的日志记录

## 系统要求

- Windows 10/11
- Chrome浏览器 或 Microsoft Edge浏览器
- 至少2GB可用内存

## 使用方法

### 1. 运行程序

双击 `lanzou_downloader.exe` 文件运行程序。

### 2. 选择浏览器

程序启动后，按照提示选择要使用的浏览器：
- 输入 `1` 选择Chrome浏览器（默认）
- 输入 `2` 选择Edge浏览器

### 3. 设置参数

- **最大并发数**：输入希望同时下载的文件数量（建议3-5个，过多可能导致连接被限制）
- **下载目录**：指定文件下载的目标目录（默认为downloads）

### 4. 输入链接信息

程序会自动访问预设的蓝奏云链接，你也可以根据需要修改源代码中的链接地址。

### 5. 选择文件

程序获取到文件列表后，可以选择要下载的文件：
- 输入单个文件编号（如：1）
- 输入多个文件编号（如：1,3,5）
- 输入范围（如：1-5）
- 混合输入（如：1,3-5,7）

### 6. 开始下载

确认选择后，程序将开始并发下载选定的文件。

## 注意事项

### 重要提醒

1. **浏览器驱动兼容性**：
   - Chrome驱动版本：ChromeDriver 142.0.7444.175
   - Edge驱动版本：Microsoft Edge WebDriver 143.0.3650.80
   - 请确保你的浏览器版本与此驱动版本兼容，否则程序无法正常启动浏览器

2. **网络稳定性**：
   - 下载过程中请保持网络连接稳定
   - 部分蓝奏云链接可能存在访问限制，如果下载失败请稍后再试

3. **反爬虫机制**：
   - 请勿设置过高的并发数，以免触发网站反爬虫机制
   - 如遇频繁失败，建议降低并发数或暂停一段时间再试

4. **文件安全**：
   - 请仅下载合法合规的文件
   - 下载前请确认文件来源的安全性

5. **权限问题**：
   - 确保程序有写入下载目录的权限
   - 如遇权限问题，请以管理员身份运行程序

### 故障排除

1. **浏览器启动失败**：
   - 检查浏览器版本是否与驱动兼容
   - 确认浏览器是否正确安装

2. **下载速度慢**：
   - 检查网络连接
   - 尝试降低并发数

3. **无法访问链接**：
   - 确认链接有效性
   - 检查网络是否能正常访问蓝奏云

## 文件说明

- `lanzou_downloader.exe` - 主程序
- `chromedriver-win64/` - Chrome浏览器驱动文件夹
- `edgedriver_win64/` - Edge浏览器驱动文件夹
- `downloads/` - 默认下载目录（程序会自动创建）
