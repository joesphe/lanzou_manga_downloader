# 蓝奏云下载器 - 打包版本使用说明

## 文件说明

- `lanzou_feapder_gui.exe`: 主程序文件，双击即可运行
- `downloads/`: 默认下载目录（如不存在会自动创建）
- `lanzou_downloader.log`: 日志文件（如遇到问题可查看）

## 使用方法

1. **直接运行**：
   - 双击 `lanzou_feapder_gui.exe` 即可启动程序
   - 程序会自动使用内置的混淆认证信息连接到蓝奏云

2. **环境要求**：
   - Windows操作系统
   - 已安装 Microsoft Edge 浏览器（程序使用Edge内核进行网页操作）

## 功能特点

- 图形化界面，操作简单直观
- 支持获取蓝奏云文件列表
- 支持批量选择文件下载
- 显示文件名、大小、上传时间等信息
- 下载进度条显示
- 自动检测已下载文件

## 注意事项

1. **首次运行**：
   - 程序可能会被安全软件误报，请添加信任后运行
   - 首次启动可能需要一些时间来加载浏览器驱动

2. **浏览器要求**：
   - 确保系统已安装 Microsoft Edge 浏览器
   - 如果无法启动浏览器，请检查Edge是否为最新版本

3. **网络连接**：
   - 确保网络连接稳定
   - 某些网络环境下可能需要等待较长时间才能获取文件列表

## 修复说明

此版本已修复了GUI模式下"获取文件列表"按钮无响应的问题。修复措施包括：
- 修改了safe_print函数，使其在GUI模式下也能正常工作
- 添加了日志记录机制，将输出写入lanzou_downloader.log文件
- 程序现在在GUI模式下（无控制台窗口）也能正常运行
- 包含了所有必要的运行时依赖文件

## 故障排除

- 如果程序无法启动，请确认已安装 Microsoft Edge 浏览器
- 如果出现网络错误，检查网络连接
- 如果下载失败，请尝试重新获取文件列表后再下载
- 如遇问题，可查看同目录下的`lanzou_downloader.log`文件获取更多信息

## 开发信息

本程序采用PyInstaller打包，包含了所有必要的依赖库，可在没有Python环境的Windows系统上独立运行。
